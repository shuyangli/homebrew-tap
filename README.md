# shuyangli/homebrew-tap

Personal Homebrew tap for [shuyangli](https://github.com/shuyangli)'s tools.

## Install

```sh
brew install shuyangli/tap/healthbridge
```

## What's here

- **healthbridge** — read and write Apple Health data via an
  end-to-end-encrypted relay and iOS companion app.
  See [shuyangli/healthbridge](https://github.com/shuyangli/healthbridge).

## Updates

Formula bumps are opened as PRs by GoReleaser on every tagged
release of the upstream repo and are auto-merged into `main`.
